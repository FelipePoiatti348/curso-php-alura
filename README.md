# Curso PHP para Desenvolvimento Web – Alura

## Descrição
Repositório com exercícios e práticas desenvolvidas durante o curso de PHP da Alura, com foco em desenvolvimento web, back-end e lógica de programação.

## Conteúdos abordados
- Sintaxe básica do PHP
- Estruturas de controle (condicionais e loops)
- Funções
- Orientação a Objetos (OOP)
- Manipulação de arquivos
- Integração entre PHP e HTML
- Conexão com banco de dados MySQL
- Boas práticas de desenvolvimento

## Tecnologias utilizadas
- PHP
- HTML
- MySQL
- Git e GitHub

## Objetivo do projeto
Aplicar na prática os conceitos estudados no curso, consolidando conhecimentos em desenvolvimento back-end com PHP e bancos de dados relacionais.
